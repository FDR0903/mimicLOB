mimicLOB
========================================================

mimicLOB is a package for 
    - LOB Simulation
	- High Frequency Strategy definiton
	- High Frequency Strategy backtesting
	


You can install it running conda

    conda install -c fdr0903 mimiclob

Usage:
    >>> import mimicLOB as mLOB
    >>>
